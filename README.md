# How to run the app

TO RUN IN DEV:

1. npm start (to run react server)
2. npm run electron-dev

TO RUN IN PRODUCTION:

1. npm run build (to build react into static files)
2. npm run electron

TO PACKAGE:

1. npm run build
2. electron-packager project_folder_name app_name (--asar)
